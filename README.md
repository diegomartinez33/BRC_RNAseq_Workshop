# BRC Bioinformatics Workshop: Introduction to Bulk RNA-seq Analysis

This workshop will cover common components of the RNA-seq workflow, including experimental design, working on the command line, accessing data on NCBI, quality control, read alignment, quantification, normalization, and differential expression analysis. Emphasis will be placed on conceptual understanding and computational logic, and the workshop is intended to be beginner-friendly. No prior bioinformatics experience is required.
